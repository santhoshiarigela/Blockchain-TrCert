# Blockchain-TrCert
Transfer Certificate Generation System
Overview
TrCert blockchain based Transfer Certificate Generation System which is used to generate, store, and share the transfer certificate to help educational institutions to eliminate fake candidates and provide security and efficiency. The certificate is generated by the University when a student applies for transfer from one college to another college. It allows the transfer of authentic users from one educational institution to another education institute and provides the information in a secure and tamper-resistant manner. TrCert is developed by integrating Ethereum smart contracts with IPFS (InterPlanetary File System). IPFS stores the certificates in a decentralized file system.

Objectives:
--> To eliminate fake certificates
--> Secure storage
--> Transparency
--> Security

Technologies used:

--> Ethereum
